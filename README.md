
# Grafik Data dan Tabel

Cara untuk menjalankan aplikasi sebagai berikut : 

## Installation

Install node modules untuk backend 

```bash
  npm install
```

Untuk menjalankan backend 

```bash
  nodemon backend/app
```

Install node modules untuk frontend 

```bash
  cd frontend

  npm install
```    

untuk menjalankan frontend

```bash
  npm run dev
```
